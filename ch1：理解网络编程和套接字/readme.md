# 第一章：理解网络编程和套接字
构建套接字的过程，以下四个函数均位于头文件`sys/socket.h`：（第二三章详细介绍参数和用法）
1. 创建套接字：socket()
2. 为套接字分配IP地址和端口号：bind()
3. 监听套接字：listen()
4. 受理连接请求：accept()

客户端套接字请求连接：
1. 创建客户端套接字：socket()
2. 向服务端发送连接请求：connect()

## 关于linux下的文件操作
0、1、2分别对应stdin、stdout、stderr。自定义文件描述符从3开始。
